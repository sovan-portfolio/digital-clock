Digital Clock with Date

A stylish digital clock application built with Java Swing that displays both the current time and date in an elegant interface.

Features
========

- Digital Clock: Displays time in 12-hour format with AM/PM indicator
- Date Display: Shows the current date in "Day, Month Date, Year" format
- Custom Styling :
  - Blue background
  - Black heading
  - Large green time display
  - Yellow date display
  - Dark gray clock background with border
- Responsive Design: Automatically adjusts layout for different window sizes
- Smooth Updates: Time updates every second
